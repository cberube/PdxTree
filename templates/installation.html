Installation

Bower

PdxTree may be installed using Bower: 

bower install pdx-tree

Individual file download

You may also install PdxTree by downloading either the pdx-tree.js or pdx-tree.min.js file directly from the dist directory inside the GitHub repository.

Git repository

You may also clone the entire repository if you are interested in modifying and improving PdxTree.

Required files

The only file required for PdxTree (aside from it's dependency on AngularJS) is the pdx-tree.js or pdx-tree.min.js from the dist directory.

If you have cloned the entire repo and want to work with a development version of PdxTree you will need to include all of the JavaScript files in the src directory instead.
