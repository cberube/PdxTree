Features

PdxTree is an AngularJS directive that aims to help manage the rendering of hierarchical data.

It consists of a primary pdx-tree directive, two light-weight marker directives (pdx-tree-item and pdx-tree-children), and several supporting services.

Data for PdxTree may be pre-loaded in the Angular scope or loaded over HTTP as needed.

PdxTree supports both a "nested" rendering style and a "sibling" rendering style. The nested style works well with standard HTML lists, while the sibling style works well with stylized listings and makes tree and table combinations possible.

PdxTree works to generate the HTML for the tree, but defers much of the control over the tree to the client code.
